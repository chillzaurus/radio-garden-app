# Radio Garden — Desktop App

An unofficial desktop wrapper for [radio.garden](https://radio.garden), built with Electron out of love for the project.

> **Disclaimer:** This project is completely independent and is not affiliated with, endorsed by, or associated with Radio Garden in any way. It is a non-commercial, open-source wrapper that simply packages the Radio Garden website into a desktop application. No profit is made from this project. All content, data, and streaming is served directly by Radio Garden's own servers.

---

## Features

- Lightweight desktop wrapper for radio.garden
- System tray with recent stations, sleep timer, and quick controls
- **Favorites overhaul** — custom folders with drag & drop, color tags, search bar, sort by location/continent
- **Now playing sync** — green line highlights the active station in Favorites, scrolls to it on tab switch
- **Keyboard navigation** — ↑ ↓ arrows to browse Favorites
- **Listening stats** — track time, countries, favorite stations and more in the About window
- **Discord Rich Presence** — shows what you're listening to
- Always on top, launch at startup, and more

---

## Installation

Download the latest `.exe` from [Releases](https://github.com/chillzaurus/radio-garden-app/releases) and run it. Auto-updates are built in.
